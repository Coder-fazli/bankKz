Installation Steps

  1. The plugin has been created in: /wp-content/plugins/financial-calculator/
  2. Go to WordPress Admin → Plugins
  3. Find "Financial Calculator" in the plugins list
  4. Click "Activate"
  5. Go to Settings → Financial Calculator to see usage instructions
  6. Use shortcodes in posts, pages, or widgets
